# About

Hi there 👋! I'm Eirini, an AI Engineer / Full-stack Data Scientist and a proponent of [Just-in-Time learning](https://xcancel.com/radekosmulski/status/1828564396265926967). I favour a minimalist approach to produce environmentally, economically, socially [sustainable](https://en.wikipedia.org/wiki/Sustainability) and robust software. 
It is my belief that effective computing is accomplished through humility, simplicity, efficiency, streamlined processes, and unified workflows.  

Currently I'm betting on [Deno](https://deno.com/). This unusual choice aims at teaching me new ways of thinking, [questioning the dominance of Python](https://www.youtube.com/watch?v=qvieJBbUuDk) in mainstream Data Science. Occasionally, higher effort is required to _avoid_ using Python. I recently came up with a [minimal, unified Python toolchain]({{ site.baseurl }}{% link _posts/2024-11-21-bring-it-back-to-basics.md %}) for navigating a fragmented and opinionated[^1] tooling landscape.    
Frankly, it's hard to make peace with using Python when Deno is such a joy to use. This is why I'll take it on a case-by-case basis, using whichever toolchain makes my life easier.

My GitHub handle was inspired by [Christoph Molnar](https://bsky.app/profile/christophmolnar.bsky.social)'s excellent succinct and informative read [Modeling Mindsets](https://christophmolnar.com/books/modeling-mindsets/).

I choose not to use Twitter / X[^2]. You can find me on [📣 [in]](https://www.linkedin.com/in/eirinikomninou)
